## ⚙️ Run Locally

Clone the repository:

```bash
git clone https://github.com/AYUSHTIWARI7126/nikks-ai-chatbot.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Create `.env` file:

```env
MISTRAL_API_KEY=your_api_key_here
```

Run the app:

```bash
streamlit run streamlit_app.py
```
